# Adding a venue map
1. openstreetmap.org → search the venue → zoom ~16-17 (pitch + 2-3 surrounding blocks fill the frame)
2. Position the pitch slightly off-center where the teardrop marker will sit
3. Screenshot generously — capture more area than needed; Claude crops to fit the panel
4. Send the screenshot to Claude → traced + styled volt-on-black asset comes back
5. Drop the finished file into public/venues/<venue-slug>.png
